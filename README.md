# Bubble Bobble Clone (Godot 4.6.1.stable)

## How to run:
- Open project in Godot 4.6.1.stable
- Run `Main.tscn`

## Controls (InputMap):
- `move_left` left arrow / A
- `move_right` right arrow / D
- `jump` - up arrow / W
- `fire` - space
- `start` / `restart` - space

## Implemented features:
- [x] Player movement + jump
- [x] TileMap collisions
- [x] Bubble shooting
- [x] Enemies patrol
- [x] Trap mechanic
- [x] Pop + score + HUD
- [x] Game over + restart

## Bonus:
- [x] enemies escape bubble and enter "rage" mode if not popped
